Add your environment variable to `/etc/environment` (Ubuntu/Debian distributions):

```shell
NEWENVVAR="hello"
```

Notes:
- Open new terminal and check via `echo $NEWENVVAR`
- Logout/login for non-terminal/UI-related processes
